## Data Limitations

The knowledge base does not contain verified real-time data about:

- exact current prices;
- stock availability;
- current promotions;
- salaries;
- confirmed current vacancies;
- company owner/director;
- exact delivery time for each order;
- full list of clients;
- active contracts;
- exact discounts for designers or contractors.

The assistant must not invent this information. If a user asks about these topics, the assistant should say that the current knowledge base does not contain confirmed data and suggest contacting the company manager.
